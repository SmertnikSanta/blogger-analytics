# Калькулятор метрик блогеров

## Как использовать:
1. Распакуйте архив в любую папку
2. Откройте файл index.html в браузере
3. Заполните данные для блогеров
4. Нажмите "Рассчитать все"

## Особенности:
- Можно добавить до 40 блогеров кнопкой "+"
- Для каждого параметра можно ввести до 10 значений
- Необязательные поля можно оставить пустыми
- Результаты появятся внизу страницы

## Формулы:
- ER% = (лайки+комменты+репосты) / (подписчики × 10) × 100%
- ERV = (лайки+комменты+репосты) / просмотры × 100%
